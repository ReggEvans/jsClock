# jsClock

jsClock is just a quick little project creating an analog clock displayed on the page using a set interval. The project was completed with vanilla javascript.

## Screenshot
![jsClock Screen Shot](images/clock.gif)

## Learning

This project was part 2 of the Javascript30 series by Wes Bos.

## Built With

Vanilla Javascript

## Installation
```
git clone https://github.com/ReggEvans/jsClock.git
cd jsClock
open index.html
```
